Info:
1.AR--人气指标 通过比较一周内的开盘价在周期价格中的高低，反映市场买卖人气
2.BR--意愿指标 通过比较一段周期内的收盘价在该周期价格波动中的地位，反映市场买卖意愿程度
3.BIAS--乖离率 通过百分比的形式来表示股价与平均移动线之间的偏离程度，多用于日线上测量股价偏离均线大小程度
4.VR--容量比率 n日收阳日成交量总和与n日收阴日成交量之和的比值，以成交量的变化确认低价和高价，从而确定买卖时法
5.pR--威廉指标 基于摆动点来量度股市的超买卖现象，可以预测循环期内的高点或低点，从而提出有效率的投资讯号
6.MACD--指数平滑异同平均线 由快、慢均线的离散、聚合表征当前的多空状态和股价可能的发展变化趋势
7.Flu--历史波动率 基于过去的统计分析得出的，假定未来是过去的延伸，利用历史方法估计波动率类似于估计标的资产收益系列的标准差
8.KDJ--随机指标 研究最高价、最低价和收盘价之间的关系，用于股市的中短期趋势分析
9.OBV--能量潮 成交量数量化，制成趋势线，配合股价趋势线，从价格的变动及成交量的增减关系，推测市场气氛
10.RSI--相对强弱指标 通过比较一段时期内的平均收盘涨数和平均收盘跌数来分析市场买沽盘的意向和实力，从而作出未来市场的走势
